# Python List String Search Methods

This Python script demonstrates multiple methods for searching for a specific string within a list. It covers basic for loops, the 'in' operator, list comprehensions, and the filter function to find string occurrences and their indices.

## Language

`python`

## How to Run

Save the code as a Python file (e.g., main.py) and run it from your terminal using: python main.py
